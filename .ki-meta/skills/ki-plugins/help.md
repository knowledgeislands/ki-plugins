# ki-plugins

Audit, conform, and scaffold a Knowledge Islands **plugin-marketplace** repo — the generated Claude plugin marketplace that projects the harness's skills and agents onto the Cowork surface (`knowledgeislands/ki-plugins`, `ADR-KI-HARNESS-002`).

**Invoke:** `ki-plugins audit <repo> | conform <repo> | help | educate <repo> | refresh`

**Modes:**

- `AUDIT`
- `CONFORM`
- `EDUCATE`
- `HELP` — explain this skill and stop; the default when no mode is given (then routes, if interactive)
- `REFRESH`

**See also:** For GitHub config and LICENSE use `ki-repo`; for Markdown/TOML style use `ki-authoring`.
